# attractorSZ

Models are adapted from [Stein, Barbosa et al. (Nat Commun, 2020)](https://www.nature.com/articles/s41467-020-18033-3) and [Hansel & Mato (J Neurosci, 2013)](https://www.jneurosci.org/content/33/1/133.short).
